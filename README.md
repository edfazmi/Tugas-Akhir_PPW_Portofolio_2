## ğŸ’¡ Instalasi dan Penggunaan

Website portofolio ini tidak memerlukan proses instalasi yang rumit karena hanya terdiri dari file statis berbasis HTML, CSS, dan JavaScript. Untuk menjalankan proyek ini, pengguna cukup memastikan bahwa aplikasi Git telah terpasang di komputer. Langkah pertama adalah melakukan clone terhadap repository menggunakan perintah git clone https://github.com/edfazmi/Tugas-Akhir_PPW_Portofolio_2. Setelah proses cloning selesai, buka folder hasil clone tersebut dengan perintah cd Git_Portofolio. Di dalam folder ini terdapat file utama bernama index.html yang menjadi halaman awal website. Pengguna cukup membuka file index.html tersebut menggunakan browser seperti Google Chrome, Mozilla Firefox, atau Microsoft Edge untuk melihat tampilan lengkap dari website portofolio ini. Semua elemen yang ada akan langsung ditampilkan tanpa perlu menjalankan server tambahan atau konfigurasi khusus.
